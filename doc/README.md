---

## Описание проекта

Новостной сайт Ингушетии — это современный информационный портал, посвящённый событиям республики, культуре, истории и ингушскому языку. Платформа создаётся как удобное цифровое пространство для публикации актуальных новостей, образовательных материалов и контента, направленного на сохранение национальной идентичности.

Сайт сочетает в себе функции классического новостного портала и тематической культурно-образовательной платформы. Помимо новостной ленты, проект включает разделы, посвящённые культурному наследию, историческим материалам и развитию интереса к ингушскому языку.

Это не просто новостной ресурс, а цифровая площадка, объединяющая информирование, культурное просвещение, историческую память и языковое наследие Ингушетии.

Проект разрабатывается с помощью AI-ассистированного подхода (Claude Code) — без традиционного программирования, в рамках бренда WEZARIO.

## Цель проекта

Создать качественный и удобный медиаресурс, который:

- освещает актуальные новости Ингушетии;
- знакомит аудиторию с культурой и историей народа;

- делает национальный контент доступным, современным и визуально привлекательным.

## Основные разделы сайта

### 1. Новости

Ключевой раздел сайта с актуальными материалами о событиях в Ингушетии, России и мире.

Содержимое: региональные новости, общественно-политические события, экономика и социальная сфера, происшествия, интервью и мнения, важные объявления.

### 2. Культура

Раздел, посвящённый традициям, искусству и культурной жизни ингушского народа.

Содержимое: народные традиции и обычаи, праздники и обряды, литература, поэзия, музыка, современная культурная жизнь, известные деятели культуры, обзоры мероприятий и выставок.

### 3. История

Материалы об историческом пути ингушского народа и ключевых этапах развития региона.

Содержимое: древняя история, исторические события, выдающиеся личности, башенная архитектура, этнография и быт, аналитические и просветительские статьи.

### 4. Ингушский язык

Образовательный и просветительский раздел, направленный на поддержку и популяризацию ингушского языка.

Содержимое: уроки и базовые материалы, грамматика, разбор слов и выражений, этимология и корни слов, тексты с переводами, материалы для начинающих и для носителей, желающих углубить знания.

## Основной функционал

### Пользовательская часть

Сайт предоставляет пользователю удобный доступ к контенту и понятную структуру навигации:

- просмотр ленты новостей;
- переход по категориям и тематическим разделам;
- просмотр отдельных статей;
- поиск по материалам;
- фильтрация по темам и тегам;
- адаптивное отображение на компьютере, планшете и смартфоне;
- удобная типографика для комфортного чтения.

### Админ-панель

Центральный инструмент наполнения сайта, через который можно добавлять все новости и весь тематический контент.

```
/admin
├── Список всех публикаций
│   ├── Быстрый поиск по статьям
│   ├── Фильтр по статусу (черновик / опубликовано)
│   └── Фильтр по разделам
├── Создание и редактирование публикаций
│   ├── Поле заголовка
│   ├── Краткое описание (лид)
│   ├── Редактор основного текста (WYSIWYG)
│   ├── Загрузка обложки статьи
│   ├── Выбор категории и тегов
│   ├── SEO-поля (meta title, meta description)
│   ├── Сохранение как черновик
│   └── Публикация / снятие с публикации
├── Управление разделами
│   ├── Новости
│   ├── Культура
│   ├── История
│   └── Ингушский язык
├── Управление главной страницей
│   └── Порядок вывода материалов
└── Удаление материалов
```

**Возможности админ-панели:**

- добавление новых новостей;
- редактирование существующих публикаций;
- удаление материалов;
- создание и редактирование статей во всех разделах (Культура, История, Ингушский язык);
- загрузка изображений к публикациям;
- выбор категории и тегов;
- сохранение материалов как черновиков;
- публикация и снятие с публикации;
- управление главной страницей и порядком вывода материалов.

## Визуальная концепция

Сайт должен выглядеть современно, строго и удобно. Основной акцент — на читаемость, доверие к информации и чистую подачу контента.

Особенности интерфейса: аккуратная новостная лента, крупные и понятные заголовки, минималистичный дизайн, чёткое разделение рубрик, боковые блоки для важных или популярных материалов.

### Цветовая схема (правило 60-30-10)

| Доля | Роль | Цвет | Применение |
|------|------|------|------------|
| **60%** | Нейтральный | Белый / светло-серый | Фон страниц, карточки, пространство |
| **30%** | Дополнительный | Синий (#1d4ed8) | Ссылки, заголовки, навигация, кнопки |
| **10%** | Акцентный | Красный (#dc2626) | Срочные новости, важные метки, badge «Срочно» |

Текст — тёмный на белом фоне для максимальной читаемости.

## Структура базы данных (Supabase)

```sql
-- Публикации (новости, статьи по всем разделам)
articles (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title           TEXT NOT NULL,
  slug            TEXT UNIQUE NOT NULL,
  lead            TEXT,                                -- краткое описание
  content         JSONB,                               -- контент из WYSIWYG (Tiptap JSON)
  cover_image     TEXT,                                -- URL обложки
  category_id     UUID REFERENCES categories(id),
  author_id       UUID REFERENCES users(id),
  status          TEXT DEFAULT 'draft',                -- draft | published
  views           INTEGER DEFAULT 0,
  is_featured     BOOLEAN DEFAULT false,               -- закреплено на главной
  sort_order      INTEGER DEFAULT 0,                   -- порядок на главной
  meta_title      TEXT,                                -- SEO заголовок
  meta_description TEXT,                               -- SEO описание
  published_at    TIMESTAMPTZ,
  created_at      TIMESTAMPTZ DEFAULT now(),
  updated_at      TIMESTAMPTZ DEFAULT now()
)

-- Разделы и подкатегории
categories (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name            TEXT NOT NULL,
  slug            TEXT UNIQUE NOT NULL,
  parent_id       UUID REFERENCES categories(id),      -- подкатегории
  sort_order      INTEGER DEFAULT 0
)

-- Теги
tags (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name            TEXT NOT NULL,
  slug            TEXT UNIQUE NOT NULL,
  is_hot          BOOLEAN DEFAULT false                -- горячие темы в шапке
)

-- Связь статей и тегов (many-to-many)
article_tags (
  article_id      UUID REFERENCES articles(id) ON DELETE CASCADE,
  tag_id          UUID REFERENCES tags(id) ON DELETE CASCADE,
  PRIMARY KEY (article_id, tag_id)
)

-- Словарь ингушского языка
dictionary (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  word_ingush     TEXT NOT NULL,
  word_russian    TEXT NOT NULL,
  transcription   TEXT,
  audio_url       TEXT,
  category        TEXT,                                -- тематическая группа
  example_ingush  TEXT,
  example_russian TEXT,
  created_at      TIMESTAMPTZ DEFAULT now()
)

-- Пользователи (администраторы и редакторы)
users (
  id              UUID PRIMARY KEY,                    -- Supabase Auth UID
  email           TEXT NOT NULL,
  display_name    TEXT,
  role            TEXT DEFAULT 'editor',               -- admin | editor
  avatar_url      TEXT
)

-- Медиатека
media (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  url             TEXT NOT NULL,
  filename        TEXT,
  alt_text        TEXT,
  uploaded_by     UUID REFERENCES users(id),
  created_at      TIMESTAMPTZ DEFAULT now()
)
```

## Быстрый старт

```bash
# Создание проекта
npx create-next-app@latest nazranka
cd nazranka

# Установка зависимостей
npm install @supabase/supabase-js @tiptap/react @tiptap/starter-kit

# Переменные окружения
cp .env.example .env.local
# Заполнить:
#   NEXT_PUBLIC_SUPABASE_URL=...
#   NEXT_PUBLIC_SUPABASE_ANON_KEY=...

# Запуск в режиме разработки
npm run dev
```
